index / mypage の記事カード : 260 x 158 - detail の関連記事カード : 120 x 85 ------------------------------------------------------------ #} {% macro post_thumbnail(post, css_class='', width=none, height=none) %} {#- 表示するサムネイルの src を優先順位に従って決定する -#} {%- if post.thumbnail_img -%} {%- set thumb_src = url_for('static', filename='img/posts/' + post.thumbnail_img) -%} {%- elif post.default_thumb -%} {%- set thumb_src = url_for('static', filename='img/thbnails/' + post.default_thumb) -%} {%- else -%} {%- set thumb_src = url_for('static', filename='img/thbnails/system-default.jpg') -%} {%- endif -%} {% endmacro %} {# ------------------------------------------------------------ post_card : 記事一覧用の記事カード（index.html / mypage.html 共通） 編集・削除ボタンは「ログイン中 かつ 自分の記事」のときだけ表示する。 マイページは自分の記事だけを一覧するため、この条件は常に真になり 従来どおり全カードに操作ボタンが表示される。 ※ current_user / csrf_token を参照するため、 import 時に「with context」が必要。 ------------------------------------------------------------ #} {% macro post_card(post) %}
